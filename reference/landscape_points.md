# Landscape Points

Random curd neutral landscape model generated using the `NLMR` package.

## Usage

``` r
landscape_points
```

## Format

A [`raster`](https://rdrr.io/pkg/raster/man/raster.html) object.

## Source

Data generated using the [NLMR](https://ropensci.github.io/NLMR/)
package. See Sciaini et al. (2018) for details:
[doi:10.1111/2041-210X.13076](https://doi.org/10.1111/2041-210X.13076) .
